# Google Search Console Setup

Use Google Search Console to monitor search visibility, indexing, sitemap status, and search terms for Coastside Contracting LLC.

## Recommended Property Type

Use a Domain property:

```text
coastsideswfl.com
```

Domain verification is recommended because it covers:

```text
https://coastsideswfl.com
http://coastsideswfl.com
https://www.coastsideswfl.com
http://www.coastsideswfl.com
```

## Verification Methods

Recommended:

```text
DNS TXT record
```

Alternative methods:

```text
HTML file upload
HTML meta tag
Google Analytics verification
Google Tag Manager verification
```

For a Domain property, Google usually requires DNS verification through the domain provider.

## Submit the Sitemap

After verification:

1. Open Search Console.
2. Select the `coastsideswfl.com` property.
3. Go to Sitemaps.
4. Submit:

```text
https://coastsideswfl.com/sitemap.xml
```

## URL Inspection

Use URL Inspection for important pages:

```text
https://coastsideswfl.com/
https://coastsideswfl.com/services.html
https://coastsideswfl.com/contact.html
```

Also inspect new city/service pages after they are published.

## Request Reindexing

For new or updated pages:

1. Paste the full URL into URL Inspection.
2. Wait for Google to check the URL.
3. Click Request Indexing.

Use this after adding new city pages, new service pages, or major content updates.

## Search Visibility Tracking

After data appears, review:

```text
Performance > Search results
Pages
Queries
Countries
Devices
```

Track which pages get impressions for searches like:

```text
cape coral handyman
fort myers handyman
cape coral bathroom remodeling
fort myers rental property maintenance
```
